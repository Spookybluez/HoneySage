# HoneySage - Honeypot Intelligence Bot

HoneySage is a smart honeypot that captures attacker behavior and uses AI to analyze and classify threats in real time.

## Features
- Simulated vulnerable HTTP login service
- Logs attacker inputs and actions
- Uses NLP and machine learning to categorize behavior
- Can expand into automated response and alerting

## Structure
- /src/honeypot.py - Fake web login service
- /src/logger.py - Log ingestion and formatting
- /src/nlp_classifier.py - Threat classification engine
